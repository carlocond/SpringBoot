# 🔗 EsRestAPI

**EsRestAPI** è un progetto di esempio che mostra come creare una **REST API** utilizzando **Spring Boot**.  
L’obiettivo è imparare a definire endpoint RESTful, gestire richieste HTTP e scambiare dati in formato JSON.

---

## 🚀 Funzionalità principali
- Creazione di endpoint REST (`GET`, `POST`, `PUT`, `DELETE`)
- Gestione delle risorse tramite Controller e Service
- Serializzazione/Deserializzazione automatica con Jackson
- Gestione semplice di errori e risposte HTTP

---

## 🛠️ Tecnologie utilizzate
- **Java 17+**
- **Spring Boot**
- **Spring Web**
- **Spring Data JPA** (opzionale)
- **H2 Database** o **MySQL**
- **Maven**
